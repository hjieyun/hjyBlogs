EasyCode代码生成

Posted on 2020-05-19

EasyCode是IDEA的一个插件，可根据数据库的表结构，生成对应的功能代码。

安装EasyCode

在idea搜索并安装Easy Code插件

连接数据库

创建好数据库，建表后，连接数据库，点击idea的 View–>tool Windows–>database,选择数据库类型后输入数据库链接进行连接

代码生成

连接成功后，选择要生成代码的表，右键后选择EasyCode –> Generate Code

自定义生成文件

默认生成的代码不可能完全符合每个人的要求，可在easy Code插件自定义生成的文件，以下是我常用的文件
