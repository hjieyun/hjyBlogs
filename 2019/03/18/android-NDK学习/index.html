android-NDK学习

Posted on 2019-03-18 | In Android

前几天开始学习andoid的ndk开发，做的一个demo。主要参考了Young_Time的文档

环境配置

1、ndk的环境配置很简单，在Android studio中加载 NDK 和 LLDB 插件。

2、下载完成后，打开 File – Project Structure – SDK Location，然后选择 Android NDK location，点击 Select default NDK ，选择ndk路径即可。

3、打开 File – Settings – Plugins，将 Android NDK Support 后面的勾去掉，重启Android studio.

开发流程

1、创建native方法接口

public class JNITest {

public native static String get();

public native static int set(int a);

}

2、选择 build – make project 生成 JNITest.class文件， 并且进入项目的 app\build\intermediates\classes\debug\ 下找到JNITest.class文件

3、使用android命令管理工具Terminal，进入app/src/main路径，使用以下语句生成 .h文件

javah -d jni -classpath D:\environment\Workspace\Android\androidWorkspace_practice\HJY_Demo_20181227\app\build\intermediates\classes\debug com.demo.hjydemo.common.jni.JNITest

javah ： 生成头文件

-d jni ： 当前目录下创建一个 jni 文件夹

D:\ … \debug ： debug文件所在本地文件路径（路径很长，可以在项目中复制路径）

com.demo.hjydemo.common.jni.JNITest ： JNITest.java所在路径

4、 执行语句后会在 app\src\main\jni 路径生成一个c语言的头文件com_demo_hjydemo_common_jni_JNITest.h，在该路径下创建一个 c/c++ resource 文件 test.c。并从com_demo_hjydemo_common_jni_JNITest.h文件复制方法到test.c中实现其逻辑。

#include<jni.h>
#include<stdio.h>
//导入我们创建的头文件
#include "com_demo_hjydemo_common_jni_JNITest.h"


//返回一个字符串
JNIEXPORT jstring JNICALL Java_com_demo_hjydemo_common_jni_JNITest_get
(JNIEnv *env, jclass jclass){
    return (*env)->NewStringUTF(env,"This is my first NDK Application");
}


//传入数字+1
JNIEXPORT jint JNICALL Java_com_demo_hjydemo_common_jni_JNITest_set
(JNIEnv *env, jclass jclass,jint a){

return a+1;

}

5、然后在app\src\main\jni目录下创建两个.mk文件。

一个 Android.mk:

LOCAL_PATH := $(call my-dir)
include $(CLEAR_VARS)

LOCAL_MODULE := JNITest
LOCAL_SRC_FILES := test.c
include $(BUILD_SHARED_LIBRARY)

一个是 Application.mk：

APP_ABI := all

6、打开 Terminal，到 app/src/main/jni 目录下，使用 ndk-build 命令生成 so 库：

ndk-build NDK_PROJECT_PATH=. NDK_APPLICATION_MK=Application APP_BUILD_SCRIPT=Android

7、在创建的JNITest中导入so库

public class JNITest {

// 动态导入 so 库
static {
    //JNITest为生成的so文件名
    System.loadLibrary("JNITest");
}

public native static String get();

public native static int set(int a);

}
